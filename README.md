Zadanie rekrutacyjne -statystyki
polegające na analizie danych z serwisu JSONPlaceholder.

## O projekcie
Aplikacja pobiera dane o użytkownikach, postach i zadaniach, a następnie generuje statystyki:
- Aktywność użytkowników (liczba komentarzy pod ich postami).
- Globalny status wykonania zadań TODO.
- Zestawienie najpopularniejszych postów.

## Technologie
- **Python** (Pandas, Requests)
- **Streamlit** (Interfejs i Dashboard)
- **Plotly** (Wykresy interaktywne)

## 💻 Jak uruchomić lokalnie
1. Zainstaluj biblioteki: `pip install -r requirements.txt`
2. Uruchom: `streamlit run app.py`
